# Добавление шрифта (yearbook-v2)

## Зачем

`lib/text-style/fonts.ts AVAILABLE_FONTS` — single source of truth для
curated-шрифтов, которые выбирает партнёр. Но чтобы шрифт работал и в
канвасе, и в PDF, его надо завести синхронно в **4 местах**. Пропустишь
одно — шрифт либо не покажется, либо покажется в редакторе, но в PDF
приедет дефолтным (молча, без ошибки).

## 4 места — все обязательны

1. **`public/fonts/*.ttf`** — положить файл(ы) шрифта. Семейство может
   требовать несколько начертаний (regular/bold и т.п.).
2. **`app/globals.css`** — добавить `@font-face` для каждого начертания.
3. **`lib/pdf-export/font-loader.ts`** — добавить во ВСЕ три места:
   `FontKey` (тип), `FONT_FILES` (путь к ttf), `resolveKey` (маппинг имени).
4. **`lib/text-style/fonts.ts`** — добавить в `AVAILABLE_FONTS`.

## Процесс

### 1. Получить файлы и проверить лицензию

- TTF-файлы начертаний от Сергея
- ⚠️ Лицензия разрешает встраивание в PDF (embedding)? Если не уверен —
  стоп, спросить Сергея (был блокер по декор-шрифтам именно из-за лицензий).

### 2. Положить ttf в public/fonts/

### 3. Прописать @font-face в globals.css

Один блок на каждое начертание, `font-display` через конвенцию проекта.

### 4. Завести в font-loader.ts (3 точки)

`FontKey` + `FONT_FILES` + `resolveKey` — все три, иначе PDF не подхватит.

### 5. Добавить в AVAILABLE_FONTS

Чтобы партнёр увидел шрифт в выборе (`resolveFontFamily` каскад).

### 6. Проверить оба пути

- [ ] Канвас редактора показывает шрифт
- [ ] PDF-экспорт встроил шрифт (открыть PDF, проверить что текст в нём,
      а не дефолт)
- [ ] `npx vitest run` / `tsc` / `next build` зелёные

## Грабли проекта

- Добавил в `AVAILABLE_FONTS`, но забыл `font-loader.ts` → в редакторе видно,
  в PDF дефолт. Молча, без ошибки — легко не заметить.
- Забыл `@font-face` → не показывается даже в канвасе.
- Лицензия не разрешает embedding → юридический риск в готовом PDF.

## Текущее состояние

7 curated-семейств, 12 TTF: Noto Serif, PT Serif, Open Sans, Roboto,
Montserrat, Caveat, Slimamif. См. `lib/text-style/fonts.ts`.

## Анти-паттерны

- Завести шрифт в 1-2 местах из 4 и считать готовым
- Добавить шрифт без проверки лицензии на embedding
- Проверить только канвас, не проверив PDF
