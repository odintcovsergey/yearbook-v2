## Что делали

ТЗ «галочка персональный раздел + нарезка альбома на per-student книги»
(19.06.2026). Часть students-секций должна сохраняться отдельно по каждому
ученику (тонкая личная книга), часть — оставаться общей (в каждом альбоме одна и
та же). Добавили признак секции `is_personal`, разметку личных страниц движком и
МОДЕЛЬ нарезки/нумерации/корешка.

**Границы ТЗ (отдельные задачи позже):** сам РЕНДЕР файлов в JPG/PDF и выгрузка в
папку — отдельное ТЗ (экспорт). Адаптация/масштабирование под формат — отдельно.
Здесь только МОДЕЛЬ: признак, логика разнесения по книгам, нумерация, корешок.

## Продуктовое решение (с Сергеем)

Персональная книга режется по физическим разворотам (печатным листам). Галочка
«персональный раздел» доступна в режимах, где на листе ОДИН ребёнок: **«1 на
страницу» (page)**, **«1 на разворот» (spread)**, **«несколько разворотов на
одного» (multi_spread)**. В режиме **«сетка» (grid)** галочки нет — на листе
разные дети, нарезать нельзя; сетка всегда общая (книга 000).

### 3. Модель нарезки + нумерация + корешок (lib/album-split/)
- **lib/album-split/types.ts**: `Book` / `BookSpread` / `AlbumBooks`.
- **lib/album-split/index.ts**:
  - `splitIntoBooks(spreads)` — разносит развороты по книгам по метке
    `personal`. Регруппировка на уровне СТРАНИЦ: в page-режиме левая и правая
    страница одного листа могут уйти в РАЗНЫЕ книги, затем страницы каждой книги
    заново собираются в развороты. Книга 000 (общая) всегда есть; личные книги
    00X по возрастанию номера ученика.
  - Нумерация: `formatBookId` (000/00X, 3 знака), `formatSpreadPosition`
    (2 знака), `bookFileName` → «КНИГА-РАЗВОРОТ» (000-01, 001-00…). Обложка = 00.
  - Корешок: `spineMmForBook` (по числу разворотов одной книги) и
    `assembledStudentSpineMm` (ФИЗИЧЕСКИ собранная тонкая книга = общие 000 +
    личные ученика). Используют существующий `resolveSpineMm` (профиль
    типографии).
- Номер книги ученика = его порядковый номер (student_index+1). Порядок учеников
  — как в движке (build-album-input: class, full_name), согласован со сводкой
  обложек (full_name) для одноклассного альбома.
