## Принципиальные решения

### x-admin-secret legacy — НЕ удаляем

Спека фазы 0.14 упоминала «чистку legacy x-admin-secret». При разведке
выяснилось:

```
lib/auth.ts:174-182        — getAuth(), Legacy режим
app/api/auth/route.ts:259  — bootstrap создания первого superadmin'а
```

Удаление обеих точек требует:
1. Проверить что никакой prod-клиент не использует `x-admin-secret`
   (фронт/curl-скрипты/internal tools)
2. Решить как создавать первого superadmin'а в новой среде без
   legacy hack — либо seed скриптом, либо изначально через UI
   (но UI требует логина, замкнутый круг)

Это **отдельная задача с миграцией** — не финальный smoke
существующей фазы. В контексте v43 фиксируем как «отложено» с
описанием почему. Технического долга на ноль не уменьшаем, но и
поспешно не ломаем.

### Шаг 1. Финальный браузерный smoke (Сергей делает)

Это **необязательный** шаг — Сергей уже видел work-side в скринах
1.4. Если хочется добить полный smoke до v43, прогнать оставшиеся
проверки на тестовом альбоме «Красночетайская СОШ»:

#### Smoke-чеклист

1. **«Скопировать JSON»** — клик в result-блоке. Должен быть toast
   «Layout JSON скопирован». Открыть Notes/любой текстовый редактор,
   вставить (Cmd+V). Должен быть валидный JSON с полями
   `layout_id`, `spreads` (массив 11 элементов), `warnings` (массив
   с `level`/`source`), `summary`.

2. **«Пересобрать»** — клик. Кнопка переходит в «Сборка...»
   disabled на 1-3 сек. После возврата result-блок обновлён, в
   `album_layouts` SQL запросом `SELECT updated_at FROM album_layouts
   WHERE album_id = '3fccf1dc-3e3d-402b-8c0c-7036e3496418'` —
   timestamp обновился.

3. **Hover на disabled кнопке** в альбоме без пресета («Школа 89»
   до того как ему назначили пресет в 1.4 smoke — но он уже назначен,
   возможно нужен другой альбом, например, «Школа 17» или «Школа 11»).
   Tooltip «Сначала выберите пресет вёрстки в форме редактирования»
   должен появиться.

Эти три проверки **не блокируют** 1.5. Можно прогнать их в любой
момент после деплоя 1.4 — и если всё ок, продолжаем со Шагом 2.
