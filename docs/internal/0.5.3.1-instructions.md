**Контекст** определить локально в файле (не в types.ts, чтобы не

type PresetBuildContext = {
  input: AlbumInput;

### 3.2. Что реализуется в 0.5.3.1

**Полностью реализовано:**

1. **Validation**: если `input.template_set_id !== templateSet.id` →
   `throw new Error(...)` (как в текущем build.ts:42).
2. **Инициализация контекста.** `cfgType = presetSlugToConfigType(preset.slug)`,
   `printType = preset.print_type`.
3. **Empty students guard**: если `input.students.length === 0` →
   warning `students_empty` + ранний return (как в build.ts:71-77).
4. **Intro section.** Если `preset.config.intro_section !== null` И
   `printType === 'soft'` (для безопасности — в seed для layflat intro=null,
   но добавим условие явно) → вызвать `buildIntroSection(ctx)`.

   Логика берётся из текущего `build.ts:buildIntroSection` (строки 933-965).
   Адаптации:
   - `applies_to_config` = `cfgType` (не из ctx.config)
   - `expected_name_hint = 'S-Intro'` зашит как раньше
   - `page_role = 'intro'`, `slot_capacity_min = { photos_full: 1 }`
5. **Teacher section.** Если `preset.config.teacher_section !== null` И
   `preset.config.teacher_section.enabled === true`:
   - Если `input.head_teacher === null` → warning `no_head_teacher` + skip
   - Если `teacher_section.layout === 'two_page'` →
     `buildTeacherSectionTwoPage(ctx)` (логика из текущего
     `buildTeacherSection`, build.ts:981-1058)
   - Если `teacher_section.layout === 'one_page'` →
     `buildTeacherSectionOnePage(ctx)` (логика из текущего
     `buildMiniSoftTeacherSection`, build.ts:1085-1131)

**Заглушки (TODO для следующих подэтапов):**

6. **Student section dispatcher** по
   `preset.config.student_section.base_layout_mode`:
   - `'single_page_per_student'` → `buildSinglePagePerStudent` *(TODO 0.5.3.2)*
   - `'spread_per_student'` → `buildSpreadPerStudent` *(TODO 0.5.3.2)*
   - `'grid_multiple_students'` → `buildGridStudents` *(TODO 0.5.3.3)*

   Каждая заглушка пишет warning:
   ```typescript
   pushWarning(ctx, {
     code: 'master_not_found',
     detail: 'TODO: <function name> not yet implemented (0.5.3.X)',
   });
   ```
7. **Thumbnails section.** Если
   `preset.config.student_section.thumbnails_section?.enabled === true` И
   `base_layout_mode !== 'grid_multiple_students'` (в Индивидуальном после
   личных разворотов; в Лайт/Мини thumbnails не имеет смысла) →
   `buildThumbnailsSection` *(TODO 0.5.3.3)*. Пишет тот же TODO warning.

8. **Return** `{ spreads: ctx.spreads, warnings: ctx.warnings }`.

### 3.3. Helper functions внутри файла

#### `presetSlugToConfigType(slug: string): ConfigType`

```typescript
const VALID_CONFIG_TYPES: ConfigType[] = [
  'standard', 'universal', 'maximum', 'medium',
  'light', 'mini', 'individual', 'tryumo',
];

function presetSlugToConfigType(slug: string): ConfigType {
  const base = slug.replace(/-(layflat|soft)$/, '');
  if ((VALID_CONFIG_TYPES as string[]).indexOf(base) < 0) {
    throw new Error(
      `presetSlugToConfigType: cannot derive ConfigType from slug='${slug}', base='${base}'`,
    );
  }
  return base as ConfigType;
}
```
