# Таблица автоверстки общего раздела OkeyBook

**Файл:** `album-autoverstka-okeybook.xlsx`
**Источник:** Сергей Одинцов (OkeyBook), 19.05.2026
**Контекст:** обсуждение фазы РЭ.21.8 — Section Structure engine

## Что это

Эталонная таблица OkeyBook, описывающая как должен вёрстаться общий раздел
выпускного альбома в зависимости от комплектации, типа листов и количества
учеников в заказе.

Это **внутренний стандарт OkeyBook**, не отраслевой. Партнёры могут хотеть
другое — таблица служит дефолтным алгоритмом, который позже можно настраивать
под конкретных партнёров.

## Структура входов

- **Комплектация:** Мини / Лайт / Медиум / Стандарт+Универсал / Максимум.
  Индивидуальная отсутствует — использует логику Максимум (решение Сергея,
  19.05.2026).
- **Тип листов:** плотные / мягкие.
- **Количество учеников в заказе** — определяет переходную страницу между
  личным и общим разделом (когда последняя страница личного раздела не
  заполнена на 100%).

## Структура выходов

### Что находится на развороте, если там заканчивается личный раздел
Колонки «левая страница» и «правая страница» — переходная страница(ы).
Пример: для Лайт плотных 13-15 учеников левая = «до 3х фото учеников +
снизу 1 общая», правая = «в приоритете 2 фото 1/2 класса, либо 6 фото 1/6,
либо 1 общая».

Эти страницы доделываются комбо-мастерами с вариациями (если не хватило
1/2 класса — берём 6×1/6 или 1×общая, и т.д.).

### Обязательный общий раздел
До 6 страниц (3 разворота). Жёсткая последовательность, входит в стоимость
альбома, всегда строится.

### Дополнительный общий раздел
До 5 страниц. Внешне НЕ отличается от обязательного — это просто платная
допуслуга OkeyBook для увеличения конверсии. Партнёр выбирает количество
разворотов в редакторе альбома (поле `common_section_max_spreads`).

## Логика «либо то, либо это»

Формулировка «либо 6 фото 1/6, либо 2 по 1/2 класса, либо 1 общая» = engine
сам выбирает на основе того что есть в пуле партнёра. Жадно по крупности
с альтернативами:
1. Есть 6 фото `sixth`? → `J-Collage-6`.
2. Иначе есть 2 фото `half_class`? → `J-Half`.
3. Иначе есть 1 фото `full_class`? → `J-Full`.
4. Иначе — страница пропускается с warning.

Таблица будет зашита константой в `lib/rule-engine/album-structure-okeybook.ts`
в подэтапах РЭ.21.8.9..12 (см. контекст-файл `yearbook-context-v87.md`).
