# ТЗ дизайнеру — коллажные страницы личного раздела («Аква меч»)

**Дата:** 17.06.2026

## Зачем
Появился режим «1 ученик на несколько разворотов»: 1 парадная страница (портрет+ФИО+цитата)
+ дальше страницы только с фото ученика. Система сама подбирает раскладку под число фото.
Сейчас вариантов раскладки мало → подбирать не из чего.

## Что уже есть в наборе (НЕ трогать)
- `E-Standard-Left` — портрет + ФИО + цитата (это и есть парадная страница).
- `E-Standard-Right` — **3 фото, без портрета** (уже готовая коллаж-страница на 3 фото).
- `E-Universal-Left/Right` — портрет + 2 фото + ФИО + цитата.

## Чего не хватает — нарисовать
Коллаж-страницы с **другим числом фото** (стиль — как у правой E-Standard, где уже 3 фото):

| Имя мастера   | Фото на странице |
|---------------|------------------|
| `E-Collage-2` | 2                |
| `E-Collage-4` | 4                |
| `E-Collage-6` | 6                |

(на 3 фото уже есть; `E-Collage-5` — по желанию, лишним не будет)

## Правила (иначе не загрузится)
1. Имя мастера — латиницей, ровно как в таблице.
2. Каждая фото-рамка (Rectangle) с меткой: `studentphoto_1`, `studentphoto_2`, … по числу фото.
3. **Только фото.** Без портрета, без имени, без цитаты — иначе страница не считается коллажной.
4. Одна страница (не разворот). `-Left`/`-Right` версии не нужны — отступ от корешка ставит система.
5. Декор (ленты/рамки-картинки) — у каждого ОБЯЗАТЕЛЬНА метка (`studentphoto_1__under` и т.п.),
   иначе пропадёт при сборке.
