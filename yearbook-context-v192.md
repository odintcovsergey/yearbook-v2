## Контекст задачи

ТЗ «стили текста в редакторе обложки (+ фикс заглушки декора)» — маленькое,
завершает редактор обложек. Два пункта: (1) подключить готовую TextStylePanel к
полностраничному редактору обложек; (2) убрать зашитую текстовую заглушку из
декор-меток.

## ✅ DONE (этот инкремент)

### 1. Панель стилей текста в редакторе обложек
`app/app/album/[id]/cover/page.tsx`:
- Импорт `TextStylePanel` + парсеры из `@/lib/text-style`
  (parseFontSizeMult/parseColor/parseHAlign/parseVAlign/parseFontFamily).
- Новое состояние `textStylePanel: { label, rightEdge, topEdge, leftEdge } | null`.
- `onTextClick` холста теперь ловит границы слота и открывает панель параллельно
  с инлайн-редактором текста (как в редакторе разворотов).
- `handleTextStyleChange` пишет служебные ключи `__fontSize__/__color__/__halign__/
  __valign__/__font__<label>` в **scope='type'** (шаблонные правки на тип обложки,
  через тот же `setTypePatches` + дебаунс `cover_save_edit`). undefined=не трогаем,
  null=удаляем ключ → кнопка «По умолчанию» работает.
- Панель закрывается при submit/cancel текста, клике по фото-слоту (кроп),
  переключении обложки в ленте, и явной кнопкой «Готово»/Esc.
- Рендер `<TextStylePanel>` читает текущие override'ы из слитых `data`,
  `templateFontFamily` — из placeholder.font_family мастера.
- Кроп портрета per-student НЕ тронут (остался как был).

### 2. Фикс заглушки декора
`lib/cover/assemble.ts` (`fillCoverData`):
- Добавлен `isDecorationLabel(label)` (метки `__under`/`__over`/`__fg[_n]`).
- В сборке такие метки явно гасятся в `null` (ключ присутствует) → холст
  (AlbumSpreadCanvas) НЕ подставляет `default_text` как заглушку.
- Решает баг: `cover_student_name__under` показывал зашитое из IDML
  «Фёдорова Варвара».
- **Почему в сборке, а не в парсере:** preview-svg уже игнорирует default_text
  (только холст подставлял его при отсутствии ключа). Фикс в `fillCoverData`
  чинит и СУЩЕСТВУЮЩИЕ обложки в БД без перезалива IDML, и будущие, и не трогает
  общий холст/редактор разворотов. Тест добавлен в assemble.test.ts.
